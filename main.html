Summary: Latitude vs. X

How does distance from the equator affect temperature. To answer this question we randomly selected 700 unique cities around the globe and then plotted their maximum temperature vs latitude for the date March 20, 2018. A random latitude and longitude were selected using the python random library. The python library citipy was used to find the city closest to those coordinates. Once the cities were selected, the OpenWeatherMap API was used to get the weather information for each city. In addition to plotting the cities’ latitudes vs temperature, we also plotted the cities’ latitude vs windspeed, humidity and cloudiness. We wanted to know if these parameters were also affected by the distance from the equator.
